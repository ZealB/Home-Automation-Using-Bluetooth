There are 3 main files;
1. Project Report: This file gives the basic idea of the project overall. 
   Contains list of components used as well as the circuit diagrams.
2. bluetooth_v2.apk: This is the android app that was built for the project. Download the App on 
   your android smartphone. Open apk to install the file. Modify phone settings to allow installation
   of 3rd party apps. Go to Home>Settings>Security>Allow_Unknown_Sources on your device.
3. main.c is embedded code through which MCU connects to the device and performs mentioned functions.
   You can use code composer studio or TI LaunchPad to burn this code into the MCU.
